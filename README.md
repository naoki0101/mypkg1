# ROS2でのサービス実装
　人の名前を送ったらその人の年齢を返すサービス

## 実行例
　＊ ros2 launch mypkg talk_listen.launch.py
     　
  ＊  [listener-2] [INFO] [1703809975.434389695] [listener]: age: 20
     
     のように実行できます。

## 必要なソフトウェア
  ＊ ros2を使える環境が必要になります。

## テスト環境
　＊　ubuntu-20.04
